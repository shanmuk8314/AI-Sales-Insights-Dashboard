# API Endpoints

## POST /api/upload

Purpose:
Accept sales CSV files from the administrator.

Request:
CSV File

Response:
Upload status and parsed data summary.

---

## GET /api/insights

Purpose:
Retrieve generated insights.

Response:

* Top Products
* Declining Products
* Weak Regions

---

## GET /api/dashboard

Purpose:
Retrieve dashboard summary statistics.

Response:

* Total Sales
* Product Performance
* Regional Performance
* Trend Analysis
